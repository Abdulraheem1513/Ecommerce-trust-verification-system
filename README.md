# E-Commerce Trust Verification and Result Authenticity System

## Project Description
This project is a Flask-based web application developed to enhance trust in e-commerce environments. It allows users to verify seller trustworthiness and report suspicious or fraudulent activities. The system aggregates reports and presents trust-related information to users in a clear and simple manner.

## Key Features
- Seller trust score verification
- Scam reporting functionality
- Display of reported scam records
- Simple and user-friendly web interface

## Technology Stack
- Python (Flask Framework)
- SQLite Database
- SQLAlchemy ORM
- HTML and CSS

## System Structure
- `app.py` – Main application logic
- `templates/` – HTML templates
- `static/` – CSS styles
- `instance/` – SQLite database storage
